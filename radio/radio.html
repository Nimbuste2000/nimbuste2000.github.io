🌐 Technologies Radio

📡 FM (Modulation de Fréquence)
La bande FM est la plus utilisée pour la radiodiffusion audio de qualité.
Fréquence : 87.5 - 108 MHz
Bande passante : 200 kHz
Zones : Mondiale
Portée : ~80-150 km
Qualité audio : Excellente (stéréo)
📻 AM (Modulation d'Amplitude)
La bande AM est plus ancienne mais offre une meilleure propagation à longue distance.
Fréquence : 531 - 1602 kHz
Bande passante : 10 kHz
Portée : Très longue (1000+ km la nuit)
Qualité audio : Moyenne (mono)
Utilisation : News, sport
📡 RDS (Radio Data System)
Système de transmission de données sur les stations FM pour afficher le nom, l'artiste, etc.
Fréquence porteuse : 57 kHz (FM)
Débit : 1187.5 bps
Données : Nom station, chanson, infos
Avantage : Navigation automatique des stations
Compatibilité : Récepteurs modernes
📢 DAB+ (Radio Numérique)
La radiodiffusion numérique terrestre offre une qualité supérieure et plus de stations.
Fréquence : 174 - 240 MHz
Codec : MPEG-4 HE-AAC
Débit : 64-256 kbps
Stations/canal : 5-14+
Zones : Europe, Asie, Afrique
🛰️ Bandes Spécialisées
D'autres bandes radio pour usages spécifiques (aviation, maritime, amateur).
VHF Aviation : 118-137 MHz
Maritime : 156-162 MHz
Amateur (Ham) : 80-2 mètres
Météo/Alerte : 162.4-162.55 MHz
PMR446 : 446 MHz
🔐 Cryptage & Sécurité
Certains signaux radio sont cryptés pour des raisons de sécurité ou de confidentialité.
Régulation : Lois nationales
P25 / DMR : Radio professionnelle
Écoute légale : Varies by country
OSINT : Analyse des métadonnées publiques
Légalité : À vérifier localement

🔐 Outils & Techniques OSINT Radio

1. RadioReference.com

Base de données mondiale des fréquences radio. Permet de rechercher par pays, ville, type de station. Très utile pour trouver les fréquences locales et analyser les infrastructures radio.

2. SDR (Software Defined Radio)

Utilisez des dongles RTL-SDR ( ~15€) avec des logiciels comme GQRX, SDR#, ou CubicSDR pour recevoir et analyser les signaux radio. Outils puissants pour l'exploration OSINT radio.

3. WebSDR & Radio en ligne

Accédez à des récepteurs SDR hébergés en ligne (WebSDR.org). Explorez les fréquences du monde entier sans matériel, parfait pour le research OSINT.

4. Analyse de cartes de couverture

Les stations publient souvent leurs cartes de couverture. En analysant ces données géospatiales, on peut identifier les zones de déploiement, les sites d'émission, et les habitudes de transmission.

5. Monitoring des fréquences d'urgence

Les signaux des services d'urgence (non cryptés dans certains pays) peuvent être monitoring légalement. Outils : Broadcastify.com, RadioGarden.com. À utiliser avec respect de la légalité locale.

6. Intelligence des métadonnées RDS

RDS contient des données publiques : nom station, format, texte radiophonique (RT). Analyser ces flux révèle les changements de programmation et l'infrastructure radio locale.

7. Osmocom Tools & Wireshark

Outils avancés pour capturer, analyser et décoder les protocoles radio. Utilisés par les chercheurs en sécurité pour identifier les vulnérabilités.

8. Géolocalisation par Triangulation RF

En mesurant la force du signal depuis plusieurs points, on peut trianguler la position d'une station d'émission. Technique classique en OSINT radio avancé.
